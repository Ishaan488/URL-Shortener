# URL Shortener

A simple web app to shorten long URLs.

## Features

- Shortens long URLs  
- Redirects to original URLs   

## How to Use

1. Clone the project:
   ```bash
   git clone https://github.com/Ishaan488/URL-Shortener.git
   cd url-shortener
   ```

2. Install dependencies:
   ```bash
   npm install
   ```
3. Enter your MongoDB connection string in "server.js" file.
    
4. Run the server:
   ```bash
   node server.js
   ```

5. Visit in browser:
   ```
   http://localhost:3000
   ```
